## 📌 Project Overview

KernelMind analyzes Linux kernel/system logs and explains OS-level problems using AI.

The system detects issues such as:

- Kernel Panic
- Deadlock
- Segmentation Fault
- Memory Issues

and provides beginner-friendly explanations and possible solutions.

---

## ✨ Features

✅ Kernel panic detection  
✅ Deadlock analysis  
✅ Memory issue detection  
✅ AI-generated explanations  
✅ Severity classification  
✅ Log file upload
